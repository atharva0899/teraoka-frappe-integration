# Teraoka Integration

Consolidated POS Data Integration for Teraoka Machines. Automates SFTP file ingestion, daily aggregation, and ERPNext data synchronization.

## Overview
This app provides:
- Automated fetching of POS files.
- Daily aggregation and chunking of sales records.
- Automatic creation of Sales Invoices and Stock Updates.
- SFTP/FTP file archiving and detailed logging.
